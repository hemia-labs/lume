# Vue 3

## Installation

```bash
pnpm add @hemia/vue
pnpm add -D @hemia/cli
```

## Initialize

```bash
npx hemia init
# Detects Vue automatically from package.json
```

## Add a component

```bash
npx hemia add button
```

Result:
```
src/components/ui/button/
├── button.vue
└── button.variants.ts
```
